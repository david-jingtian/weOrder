# weOrder - Group Food Ordering App

A modern, clean group food ordering application built with React and Tailwind CSS.

## Features

- ✅ Create a new group order with restaurant name
- ✅ Display a sample menu with food items and prices
- ✅ Add items to personal cart
- ✅ Show each person's subtotal and the group total
- ✅ Input group size (number of people)
- ✅ Calculate and display delivery fee split equally among participants
- ✅ Show final totals for each person (items + split delivery fee)
- ✅ Modern, clean UI with Tailwind CSS

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

## Usage

1. Enter a restaurant name and click "Create Order"
2. Browse the menu and add items to your cart
3. Set the group size to split the delivery fee
4. View individual subtotals and the group total
5. See final totals including the split delivery fee

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Modern ES6+ JavaScript
